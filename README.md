# vue-admin

A Vue 3 admin UI component library with layout and UI components.

## Installation

```bash
npm install vue-admin
```

This will also install the required peer dependencies:
- `clsx`
- `tailwind-merge`
- `class-variance-authority`

## Peer Dependencies

Make sure you have `vue` installed in your project:

```bash
npm install vue
```

## Components

### Layout
- `AdminLayout` тАУ Main admin layout wrapper
- `AppHeader` тАУ Top navigation header
- `AppSidebar` тАУ Side navigation bar

### UI
- `Button` тАУ Button component with variants
- `DataCard` тАУ Card for displaying data
- `StatsCard` тАУ Statistics card
- `StatusBadge` тАУ Badge for status display
- `Icon` тАУ Icon wrapper component

### Utilities
- `cn` тАУ Class name utility (`clsx` + `tailwind-merge`)
